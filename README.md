# Greenden – Plant Store Website

Greenden is a responsive and modern plant store website built using **HTML5** and **Tailwind CSS**.  
This project is designed to showcase frontend development skills, clean UI design, and responsive layout techniques.

It includes multiple pages such as Home, Products, and Contact, giving a complete static website experience.

---

## About the Project

Greenden is a beginner-friendly frontend project created to practice:

- Multi-page website development
- Tailwind CSS styling
- Responsive web design
- Navigation bar design
- Layout structuring

The website focuses on a plant-themed design with a clean and user-friendly interface.

---

## Features

-  Attractive Home Page
-  Product Listing Page
-  Contact Page
-  Fully Responsive Design
-  Modern UI using Tailwind CSS
-  Smooth Navigation Between Pages
-  Organized Folder Structure
